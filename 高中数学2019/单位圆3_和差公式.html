正弦倍角

sin2α = sin(α + α)

        = sinα∙cosα + cosα∙sinα

        = 2sinα∙cosα


sin2α + 1 = (sinα+cosα)2


sin2α = 2tanα/(1+tan2α)

余弦倍角

cos2α = cos(α + α)

        = cosα∙cosα − sinα∙sinα

        = cos2α − sin2α

        = 1 − 2sin2α

        = 2cos2α − 1


cos2α = (1−tan2α)/(1+tan2α)
