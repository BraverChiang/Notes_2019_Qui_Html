定义

正弦: sinθ = 对边/斜边 = y/1 = y

余弦: cosθ = 邻边/斜边 = x/1 = x


正切: tanθ = 对边/邻边 = y/x

余切: cotθ = 邻边/对边 = x/y


正割: secθ = 斜边/领边 = 1/x

余割: cscθ = 斜边/对边 = 1/y
